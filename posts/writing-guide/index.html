Writing Guide (Posts)

Friday, 20 December 2024

This page documents how to write articles for this site using Markdown.

Front matter

Every post needs front matter at the top:

---
title: "Your Title"
date: 2024-12-20
language: "EN"
tags: ["example", "notes"]
---

Notes:

  • Use language: "AR" for Arabic posts.
  • The tags field is optional.

Headings

Use ## for sections and ### for subsections.

## Section Title
### Subsection Title

Links and emphasis

This is *italic* and **bold**.
This is a [link](https://example.com).

Lists

- First item
- Second item

Quotes

> This is a blockquote.

Code

Inline code uses single backticks. Code blocks use triple backticks:

Inline `code` looks like this.

```bash
echo "Hello"
```

Images

Add images with a relative or absolute path:

![Alt text](/images/example.jpg)

Footnotes

Use standard Markdown footnotes. Add the reference in the text and define it at the end:

Here is a sentence with a footnote.[^1]

[^1]: This is the footnote text.

You can add multiple footnotes:

First note here.[^1] Another note here.[^2]

[^1]: First footnote.
[^2]: Second footnote.
