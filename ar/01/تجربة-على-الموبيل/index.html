ازاي تـblog من على التليفون؟

<span title='2024-03-17 00:00:00 +0000 UTC'>مارس 17, 2024</span>&nbsp;·&nbsp;دقيقتان&nbsp;·&nbsp;393 كلمة&nbsp;·&nbsp;علي مشالي | Suggest Changes

مقدمة#

دلوقتي انا عاوز اشرح شوية حاجات بذاكرها، او عاوز اكتب افكار ليا هنا في البلوج عشان تكون متأرشفه وأقدر ارجع ليها بعد كده بسهولة بدل ما أنا بنشر على الفيسبوك وعلى تويتر.. ألخ. بس الفكرة دلوقتي انا فالجيش، وأحيانا بيكون في وقت فراغ بستغله أحيانا برده في المذاكرة، او أحيانا بقرأ كتاب فـ بلقط فكره والفكرة دي بتكون عاوزه مقال لوحده او عدة مقالات لكن بكسل لأن مفيش حد بيقرأ الحاجات اللي بشاركها. المهم، قولت لنفسي انا عاوز ابدأ تاني في التدوين أثناء الخدمة العسكرية، بالذات شرح مواد مهمة زي الـ Computer Architecture & Logic Design.. فكنت بدور على أدوات ممكن تساعدني على الموبايل بحيث أقدر أcompile ال hugo project اللي عندي.. دورت كتير لحد ما لقيت حل راضي عنه.

الأبلكيشنز الأساسية#

اول حاجة انا كنت محتاجها هو الـGit .. بعد كده سألت نفسي ازاي اقدر اخلي الGit يشتغل على التليفون.. بعد بحث كتير شوية.. لقيت app اسمه Termux.. الأبلكيشن ده لعبت فيه شوية لحد ما قدرت أcpile عليه debian image .. ونزلت عليه desktop environment اسمه xfce وقدرت من خلال الvnc اني أتحكم في الsystem اللي موجود على الـlocalhost اللي هو الموبايل. المهم، ده موضوع تاني تماما يطول شرحه.. خلاص أحنا احتاجنا الGit فلقيت الـ Termux وبالتالي قدرت انزل عليه hugo من الpackage manager الخاص بالـ Termux .. تمام زي الفل. دلوقتي انا عملت push للبروجكت اللي موجود عالكمبيوتر.. دلوقتي انا محتاج اعمل clone من على الموبايل بتاعي .. طيب، طبعا أول حاجة هتعملها هتنزل git

pkg update 
pkg upgrade
pkg install git

زي الفل، نظبط بقى الconfigurations بتاعتنا

git config --global user.name "<your username>"
git config --global user.email "<your email>"

تمام، دلوقتي عاوزين نعمل clone للـ بروجكت بتاعنا.. بس أول حاجة احنا عاوزين Termux يقدر يوصل للملفات بتاعتنا اللي على التليفون.. هنعمل ده من خلال الـcommand ده:-

termux-setup-storage

دلوقتي خلينا نـnavigate للـlocation ده:

cd storage/shared/documents

هنعمل دلوقتي clone للـ repo بتاعتنا

git clone https://github.com/blackaly/mobile-blogging.git

Hugo#

حان الوقت الأن اننا نـcompile البروجكت بتاعنا.. اول حاجة طبعا لازم ننزل hugo، بكومند بسيط:-

pkg install hugo

…

تمام المفروض دلوقتي أنه نزل عالجهاز عندك.. اول حاجو متنساش طبعا انك تـnavigate للـroot folder بتاع البروجكت بتاعك، واللي هتعمله انك هترن الـcommand ده:-

hugo server -D --noBuildLock

المفروض يظهر لك دلوقتي الـouput ده..

والبروجكت بتاعنا شغال عــــادي..

بالمناسبه انا شغال على code editor اسمه Acode هو في مشاكل بس يعني مناسب للموبايل وكويس. بس، والمقال ده طبعا كاتبه وانا بستخدم الموبايل : )

good bye.
